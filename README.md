## ğŸˆ How to play? <a name = "play"></a>
<h3>Game Controls</h3>

- <b>Arrow Keys:</b> Move your character left, right, or up/down.
- <b>Spacebar:</b> Make your character jump.
- <b>P:</b> Pause or resume the game.
- <b>Esc:</b> Quit the game.
